INTRODUCTION
------------

Package Field provides a package file field type which accepts a zip.
It will render html, css and js of the zip loaded in the field. The zip
file loaded include index.html, main.js and style.css and should be
placed in root folder.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/package_field

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/package_field


REQUIREMENTS
------------

No special requirements.


RECOMMENDED MODULES
-------------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------

The module has no menu settings. For this include new field in any of the
content-type type, it will give a choice of include bundle document field and
will acknowledge just zip augmentation.
